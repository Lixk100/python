# 关系运算符  
| Operator |           Meaning           |
| :------: | :-------------------------: |
|    <     |        Is less than//小于         |
|    <=    |  Is less than or equal  to//小于或等于  |
|    >     |       Is greater than//大于       |
|    >=    | Is greater than or equal to//大于或等于 |
|    ==    |         Is equal to//等于         |
|    !=    |       Is not equal to//不等于       |  
终端中输入`python3`  
```python
>>> 1 < 2
True
>>> 3 > 34
False
>>> 23 == 45
False
>>> 34 != 323
True
```
